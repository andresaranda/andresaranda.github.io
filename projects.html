Projects

Online Multiplayer Chess

Realtime mobile responsive online multiplayer chess game built from scratch with no chess libraries in Javascript with socket.io and node.js. Game engine was built in back-end in Javascript with node.js and express, following principles of clean code, modulation and the MVC model. Front-end was made in vanilla Javascript, CSS and HTML utilizing grid, flexbox and ES6 principles. Front and back end comunication was done through socket.io for a realtime responsive design.

Landing Page for Specialty Grocery Store

Landing page for a real specialty grocery store made entirely in Javascript, CSS and HTML with a very mobile responsive design. ES6, CSS3 and HTML5 were utilized along with responsive menus and transitions, tailored for different screen sizes. The store's original colors, themes and logo were used as a basis for the design layout.
